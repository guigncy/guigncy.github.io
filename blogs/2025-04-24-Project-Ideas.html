Project Ideas

Linux-Focused Problem-Solving Projects:

  • Website Blocker: A script that modifies the system's host file to block access to a predefined list of websites (use with caution!).
  • Simple System Resource Monitor (Command-Line): A script that displays current CPU usage, memory usage, and disk space in the terminal.
  • Automated Backup Script for Specific Folders: A script that uses rsync or tar to automatically back up specified directories to another location.
  • Custom Command Alias Helper: A script that allows you to easily add, list, and remove command aliases in your shell configuration files.
  • Network Connectivity Checker with Notifications: A script that periodically pings a reliable website or IP address and sends a notification if the connection fails.
  • Simple File Search Tool with Regex (Command-Line): A command-line tool that takes a directory and a regular expression as input and lists matching files.
  • Clipboard History Manager (Simple Command-Line): A basic command-line tool that stores a history of text copied to the clipboard.
  • Automated Screenshot Tool with Timestamp: A script that takes a screenshot and saves it with a timestamp in the filename.
  • Simple Task Scheduler (Basic Cron Wrapper): A script that provides a more user-friendly interface to schedule basic tasks using cron.
  • Directory Size Analyzer with Visual Output (Basic): A script that calculates and displays the size of directories, potentially with a simple visual representation.
  • Temporary File Cleaner for Specific Applications: A script that targets the temporary file directories of specific applications and removes older files.
  • Simple Mount Point Checker: A script that verifies if specific mount points are correctly mounted.
  • CPU Temperature Monitor (Basic): A script that reads and displays the current CPU temperature.
  • Public IP Address Notifier: A script that retrieves your current public IP address and notifies you if it changes.
  • Wi-Fi Strength Indicator: A script that displays the current Wi-Fi signal strength.
  • DNS Server Tester: A script that tests the response time and reliability of different DNS servers.
  • URL Availability Checker: A script that takes a list of URLs and checks if they are currently accessible.
  • Simple Text File Encrypter/Decrypter: A basic tool to encrypt and decrypt text files using a simple cipher.
  • Markdown to HTML Converter (Basic): A script that takes a Markdown file as input and converts it to a simple HTML output.
  • Image Resizer (Simple): A script that takes an image file and resizes it to a specified width or height.
  • Audio Volume Control (Command-Line): A script to easily adjust the system's audio volume from the command line.
  • Simple Note-Taking App (Text File Based): A command-line tool to quickly create and view simple text-based notes.
  • Process Killer by Name: A script that takes a process name as input and kills all processes with that name.
  • Basic Webpage Downloader: A script that downloads the HTML content of a given URL.
  • Simple Link Checker for Local HTML Files: A script that parses local HTML files and checks for broken links.
  • System Uptime Tracker: A script that displays how long your system has been running since the last boot.
  • File Extension Counter: A script that counts the number of files with each extension in a given directory.
  • Simple Calculator with History (Command-Line): A command-line calculator that stores a history of previous calculations.
  • Random Wallpaper Changer: A script that randomly selects and sets a new wallpaper.
  • Basic Stopwatch/Timer (Command-Line): A simple stopwatch or timer that can be started, stopped, and reset.
  • QR Code Generator (Simple): A script that takes text as input and generates a QR code image.
